* Definition

The =amplitudes= are in general a set of singles $t_i^a$ and doubles $t_{ij}^{ab}$ amplitudes
stored as tensors.
They can be generated using the [[file:../algorithms/CoupledCluster.org][CoupledCluster]]
algorithm.
