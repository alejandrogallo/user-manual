1 Brief description¶

Read reads a cc4s tensor or set of tensors.

2 Algorithm call¶

A typical input file snippet to call the Read algorithm is given below.

- name: Read
  in:
    fileName: "CoulombVertex.yaml"
  out:
    destination: CoulombVertex
